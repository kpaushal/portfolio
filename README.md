# Paushal &mdash; Fullstack / Software Developer Portfolio

This is a simple, static portfolio site for **Paushal**, focused on **Fullstack / Software Developer (React & Node.js)** roles.

## Getting Started

- Open `index.html` directly in your browser (double-click the file or drag it into a browser window).
- No build step or dependencies are required.

## Customization

Update the content in `index.html`:

- **Hero section**: update your introduction, role, and summary.
- **Experience**: replace `Company Name`, `Location`, and `Dates` with your real information.
- **Projects**: replace project titles, descriptions, and tech stacks with your own.
- **Skills**: remove technologies you donâ€™t use and add any that are missing.
- **Education & Certifications**: replace the placeholder brackets with your real details.
- **Contact**: update email, LinkedIn, GitHub URLs, and any other links.

To enable the **Download Resume** button:

1. Export your resume as a PDF.
2. Save it in this folder as `Paushal_Resume.pdf` (or update the link in `index.html` to match your file name).

## Structure

- `index.html` &mdash; main page markup.
- `styles.css` &mdash; layout, colors, and responsive styling.
- `README.md` &mdash; this documentation.
